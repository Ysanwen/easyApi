@QueryParam

@QueryParam

查询参数设置，用法：@QueryParam 字段类型 字段名称 字段的相关描述。

可以设置多条查询参数，最终在发送ajax请求时会拼接到请求路径上。

字段类型 用{}里面包裹对应的类型来描述，一般为{String}、{Number}、{Boolean}等。 @QueryParam最终会转换成String类型，拼接到url里面。

字段名称 用来指示在ajax请求时的查询字段，比如: pageSize。当字段名称用[]包裹时，表示该字段为可选传；无任何包裹时，表示该字段为必传。

示例：

/**
 * @Api get /api/info/list
 * ...
 * @QueryParam {Number} page 此处page为必传字段
 * @QueryParam {Number} [pageSize] 此处pageSize为选传字段
 * ...
 */

上述示例，若page值为1， pageSize值为20，则产生的实际请求为/api/info/list?page=1&pageSize=20。
