@UrlParam

@UrlParam

请求路径中的参数，搭配@Api使用， 用法：@UrlParam 字段类型 字段名称 字段的相关描述。

字段类型 用{}里面包裹对应的类型来描述，@UrlParam只支持String类型，因此一般为{String}。

字段名称 需要与@Api 路径中{}包裹的字段一致，比如: @Api get /api/info/{id} 中url参数为id，则@UrlParam的字段名称也必须为id

示例：

/**
 * @Api get /api/info/{id}
 *
 * ...
 *
 * @UrlParam {String} id 与@Api中路径参数id一致
 */

此配置在发送ajax示例请求时，@UrlParam获取到id的值，将会用来替换掉请求路径中的{id}部分。 例如：若获取到id的值为1，实际的请求地址为/api/info/1。
